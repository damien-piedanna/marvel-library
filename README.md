Tools to browse the list of Marvel characters.

### Technical stack

- Nuxt 3
- Tailwind CSS

### Local installation

1. Clone the repo.
   ```sh
   git clone https://github.com/damien-piedanna/marvel-library.git
   ```
2. Set your local environment variables.
3. Install packages.
   ```sh
   yarn install
   ```
4. Launch a dev server
   ```sh
    yarn run dev
   ```
